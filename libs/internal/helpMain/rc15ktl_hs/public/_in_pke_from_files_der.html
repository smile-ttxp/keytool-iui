Import private key entry from DER file and certs chain file

Create private key entry by loading private key file in DER format, and associated certificates chain file.

Workflow

	
	Fill in all required fields
	(Make sure key algo selected in the drop list is the right one)
	  ==> enable action button located at bottom of active window
	
	Click action button
	  ==> new window shows up:
	    Window displays contents of selected keytore
	
	At bottom, enter new alias's entry and respective password, then click OK button
	  ==> that's all!
	
	
	Note: No password required for PKCS12 keystores.

Limitations

Raw (binary) files format.
