Import trusted certificate entry from file

Workflow

	
	Fill in all required fields
	  ==> enable action button located at bottom of active window
	
	Click action button
	  ==> new window shows up:
	    Window displays contents of selected keytore
	
	At bottom, enter new alias's entry, then click OK button
	  ==> that's all!

Limitations

	  Trusted certificate file should just contain one certificate.

Note about "Check root CA certs store" checkbox

	
	If checked, the certificate candidate will be compared with the certificates contained in system's root CA certificates store.
	
	Case 1) self signed certificate
		if certificate candidate already exists in root CA certificates store, you will be warned and asked to
		confirm the import of the certificate.
	
	Case 2) not self signed certificate
		a message dialog will show up in case the issuer of certificate is listed in root CA certificates store.
