Verify signed data file with trusted certificate entry and detached signature file

Workflow

	
	Fill in all required fields
	  ==> enable action button located at bottom of active window
	
	Click action button
	  ==> new window shows up:
	    Window displays contents of selected keytore
	    An additional column (first one, on the left) displays valid entries.
	
	Click on valid row
	  ==> respective alias is displayed at bottom
	  
	Click OK button
	  ==> if signature is OK, message dialog shows up 
	      else warning dialog shows up.

Limitations

	
	None.
