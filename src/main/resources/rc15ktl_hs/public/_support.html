Session log

The software generates a log of the events that occur during a session. Any warning or error caught is stored in the session log file.

Log named "session.log" is located in:

  • Unix & Linux: [user.home]/.rcp/usr/ktl/[appli.version]/[user.name]/session.log
  • Windows: [user.home]\.rcp\usr\ktl\[appli.version]\[user.name]\session.log

Application's cache dir and user's name are displayed in the Advanced tab of About KeyTool IUI dialog window.

Note: Under Java WebStart, the logs generated by the appli may be redirected either to the Java console, or in another log file, depending on User's WebStart preferences settings.  

Config log

At starting time, the software stores several system's information in a file named "config.log" located in:
  • Unix & Linux: [user.home]/.rcp/usr/ktl/[appli.version]/[user.name]/config.log
  • Windows: [user.home]\.rcp\usr\ktl\[appli.version]\[user.name]\config.log

Application's cache dir and user's name are displayed in the Advanced tab of About KeyTool IUI dialog window.
