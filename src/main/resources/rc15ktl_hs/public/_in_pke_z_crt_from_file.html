Import certificate to private key entry from C.A. cert. reply file

Workflow

	
	Fill in all required fields
	  ==> enable action button located at bottom of active window
	
	Click action button
	  ==> new window shows up:
	    Window displays contents of selected keytore
		An additional column is displayed at the left of the tables. This column shows candidate entries for this task.
	
	Click the right candidate row, the one that was used to create CSR (Certificate Signing Request) file.
	  ==> the highlighted entry's alias named is displayed at bottom
	
	At bottom, enter new alias's entry and respective password, then click OK button
	  ==> that's all!
	
	Note: No password needed for PKCS12 keystore.

Limitations

	  None.
