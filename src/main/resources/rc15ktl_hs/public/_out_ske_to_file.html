Export secret key entry as file

Workflow

	
	Fill in all required fields
	  ==> enable action button located at bottom of active window
	
	Click action button
	  ==> new window shows up:
	    Window displays contents of selected keytore
		An additional column is displayed at the left of the tables. This column shows valid entries for this task.
	
	Click a valid row
	  ==> the highlighted entry's alias named is displayed at bottom
	  
	Right below, enter respective password for this entry, then click OK button
	  ==> that's all!

Limitations

	  Keystore types:
	  . JCEKS
	  . BKS
	  . UBER
